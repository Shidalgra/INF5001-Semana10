Crea una carpeta llamada public en la raíz de tu proyecto (al mismo nivel que models e index.js).

Abre tu archivo index.js y agrega la siguiente línea de código justo debajo de las otras líneas 
que dicen app.use(...):

    app.use(express.static('public'));

Para que el navegador pueda cargar nuestra página, primero debemos
asegurarnos de que la ruta estática que creamos al inicio apunte al 
lugar correcto. Vamos a conectar los botones de la interfaz con las 
rutas del servidor (POST, GET, PUT, DELETE) usando fetch en JavaScript.

Vamos a la siguiente ruta: public/Assets/js/ y creamos un archivo que se llame:

*   CURD-MongoDB

Dentro pegamos esté código:

    /*------------------*/
    const formulario = document.getElementById('formulario-tarea');
    const inputTitulo = document.getElementById('input-titulo');
    const listaTareas = document.getElementById('lista-tareas');

    // 1. Cargar y mostrar las tareas al abrir la página (GET)
    async function obtenerTareas() {
        listaTareas.innerHTML = ''; // Limpiar la lista
        const respuesta = await fetch('/tareas');
        const tareas = await respuesta.json();
        
        tareas.forEach(tarea => {
            const li = document.createElement('li');
            li.innerHTML = `
                <span>${tarea.titulo}</span>
                <button onclick="eliminarTarea('${tarea._id}')" style="background:red; margin-left:10px;">Eliminar</button>
            `;
            listaTareas.appendChild(li);
        });
    }

    // 2. Crear una nueva tarea desde el formulario (POST)
    formulario.addEventListener('submit', async (e) => {
        e.preventDefault();
        const titulo = inputTitulo.value;

        await fetch('/tareas', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({ titulo })
        });

        inputTitulo.value = '';
        obtenerTareas(); // Recargar la lista
    });

    // 3. Eliminar una tarea (DELETE)
    async function eliminarTarea(id) {
        await fetch(`/tareas/${id}`, { method: 'DELETE' });
        obtenerTareas(); // Recargar la lista
    }

    // Ejecutar al cargar la página
    obtenerTareas();
    /*------------------*/

Una vez hecho esto en la parte de abajo del archivo index.html que está en la carpeta
public, vamos antes de el cierre de body y le damos está línea:

Mejorando el Diseño (CSS)
Para lograr un aspecto de alta calidad, podemos aplicar algunas técnicas de diseño actuales:

Fondo y Contenedor: Usar un fondo suave con un contenedor blanco que tenga esquinas redondeadas
🔲 y una sombra sutil para darle profundidad.

Tipografía y Colores: Cambiar la fuente clásica por una más limpia como 'Segoe UI' o Roboto, 
y usar una paleta de colores armónica 
(por ejemplo, tonos de azul oscuro, gris y acentos verdes o rojos).

Efectos Interactivos (Hover): Añadir transiciones suaves en los botones para que cambien de color
suavemente al pasar el mouse por encima.

Distribución de Elementos: Usar flexbox en los elementos de la lista (<li>) para que el texto de 
la tarea quede alineado a la izquierda y el botón de eliminar se mantenga fijo a la derecha de 
forma ordenada.

Aquí tienes estilos para reemplazar tu style.css:
